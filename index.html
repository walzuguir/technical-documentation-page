HTML

HTML (HyperText Markup Language) é a linguagem padrão para a criação de páginas web. Ela define a estrutura de um documento web através de uma série de elementos.

Os elementos HTML são os blocos de construção de todas as páginas da web. Eles são representados por tags e podem conter texto, links, imagens, entre outros conteúdos.

<!DOCTYPE html>
<html>
<head>
<title>Título da Página</title>
</head>
<body>
<h1>Cabeçalho Principal</h1>
<p>Este é um parágrafo.</p>
</body>
</html>
  • !DOCTYPE html: Declaração do tipo de documento, que define a versão do HTML.
  • html: Elemento raiz que envolve todo o conteúdo do documento.
  • head: Contém metadados sobre o documento, como o título.
  • body: Contém o conteúdo visível da página, como cabeçalhos e parágrafos.
CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para descrever a apresentação de um documento escrito em HTML ou XML.

Com o CSS, você pode controlar a cor do texto, o estilo das fontes, o espaçamento entre parágrafos, como os elementos são posicionados e dispostos, entre outras coisas.

body {
  color: black;
  font-family: Arial, sans-serif;
}
h1 {
  color: blue;
}
  • body: Seletor que aplica estilos ao corpo do documento.
  • color: Define a cor do texto.
  • font-family: Define a família de fontes do texto.
  • h1: Seletor que aplica estilos ao cabeçalho de nível 1.
JavaScript

JavaScript é uma linguagem de programação que permite a criação de conteúdo dinâmico e interativo em páginas web.

Ele pode ser usado para atualizar dinamicamente o conteúdo de uma página, validar dados de formulários, criar animações, entre outras funcionalidades.

document.getElementById("demo").innerHTML = "Hello JavaScript!";
  • document.getElementById: Método que retorna o elemento com o ID especificado.
  • innerHTML: Propriedade que define ou retorna o conteúdo HTML interno de um elemento.
Python

Python é uma linguagem de programação de alto nível, conhecida por sua sintaxe clara e código legível.

Ela é amplamente utilizada em desenvolvimento web, ciência de dados, automação, e muitas outras áreas.

print("Hello, Python!")
  • print: Função que imprime a mensagem passada como argumento no console.
Apex

Apex é uma linguagem de programação proprietária da Salesforce, usada para desenvolver aplicativos personalizados na plataforma Salesforce.

Ela permite aos desenvolvedores adicionar lógica comercial e interações com banco de dados nos aplicativos Salesforce.

public class MyClass {
  public static void myMethod() {
    System.debug('Hello, Apex!');
  }
}
  • public class: Declaração de uma classe pública em Apex.
  • public static void myMethod: Declaração de um método público e estático que não retorna valor.
  • System.debug: Método que imprime a mensagem passada como argumento no console de depuração.
